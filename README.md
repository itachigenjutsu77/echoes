# Echoes

Echoes is a minimalist social deduction multiplayer game inspired by Steve Jobs' design principles. The game emphasizes connection through silence and simplicity, focusing on human understanding through language.

## Core Concept
- Tagline: Connection Through Silence
- Eight players participate in rounds where they submit one-sentence responses to single-word prompts
- No kill mechanics, sabotage, or complex tasks
- Emphasizes pure social deduction via intuition and language

## Design Principles Inspired by Steve Jobs
1. Start with customer experience, not technology
2. Radical simplicity in mechanics and interface
3. Elimination of unnecessary features for clarity
4. Visual identity with austere and clean design

## Gameplay Loop
- Players receive a word prompt
- Submit a one-sentence response in 60 seconds
- Vote anonymously on the response that best matches a hidden truth
- Score based on vote accuracy

## Development Plan
- Phase 1: Build MVP with lobby, prompt system, voting, and scoring
- Phase 2: Playtest extensively with real users
- Phase 3: Polish design and optimize for launch

## Platform
- iOS/macOS using Swift with Firebase backend
- Cross-platform ready for iPhone, iPad, Mac, and web

## Why Apple Would Acquire Echoes
- Instantly shareable, highly engaging without addiction
- Clean, minimalist design aligned with Apple values
- Unique game mechanic that fosters meaningful connection
